Testy jsou rozděleny do 3 katergorií
1. korektnost vstupu - testují správný syntax a zkoušejí ruzné variace a kombinace vstupu
2. optimal - testy pro generický algoritmus- vyznačují se velkým množstvím položek
3. brutForce - testy pro hrubou sílu - vyznačují se menším počtem položek, v některých případech obsahují obskurdnější instance knapsaku
